# Dubai Mall – Interactive Sales Experience

## Overview
A cinematic, interactive web experience built to present Dubai Mall as a global destination for retail, entertainment, and brand partnerships.

The project replaces traditional static sales decks with a modern, immersive web-based storytelling experience.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer (animations)

## Features
- Fullscreen cinematic hero section
- Scroll-based interactive storytelling
- Video-driven UI experience
- Modular section-based architecture

## Setup Instructions

Clone the repository:
git clone <repo-link>

Install dependencies:
npm install

Run development server:
npm run dev
